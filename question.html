Question-1: What are the different ways to select an element in the DOM?

getElementById, getElementByClassName, getElementByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML catches the HTML content in an element innerText catches the Text content that is visible in am element but doesn't include any hidden text textContent catches the text content both that is visible and hidden

Question-3: What is event delegation in the DOM?

Event delegation means when you set a event listener on the parent element to handle the events on the child elements.

Question-4: What is event bubbling in the DOM?

Events have a default behavior where if it is triggerd it strats from the target element and moves up or bubble up to the root of the DOM. This whole process is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element we use document.createElement() To add an element we use the appendChild() To remove an element we use the .removeChild() or .remove()
